# Meal Builder

A small, installable web app for building and saving meal combinations.

## GitHub Pages setup

1. Create a new public GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. Wait for GitHub to show the published address.
7. Open that address in Chrome on Android.
8. In Chrome's menu, choose **Add to Home screen** or **Install app**.

All ongoing changes and saved combinations are kept in the browser's local storage.
Use the app's Export Backup button periodically.
